Project Introduction

The project is use Machine Learning approach to detect Vehicle based on LiDAR point cloud dataset. The mean purpose of this project is to develop a new generation Arifical Intelligence perceive environment push forward autonomous vehicles.

The major features are: real-time video streaming on HTML page, vehicle detection based on LiDAR point cloud dataset, vehicle detection on 2D video streaming, and real-time data flow streaming statistic diagram.
